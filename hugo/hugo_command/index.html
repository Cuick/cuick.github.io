Hugo基本操作命令

3/Mar 2016
Hugo

注意点

  • 创建一个新的文档后，默认是draft属性是true

常用命令

  1. 本地执行
    hugo server –theme=hugo-redlounge –buildDrafts
  2. 创建一个文档
    hugo new hugo/101.md
  3. 创建一个项目
    hugo new site blols
  4. 生成静态网站
    hugo –theme=hugo-redlounge –baseUrl=“http://cuick.github.io/"
